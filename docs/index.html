Deploy model

Production Deployment

Backend production dùng Docker Compose theo từng lớp để nhiều project có thể dùng chung hạ tầng trên cùng server.

deploy/infra

Chạy shared MySQL và shared Redis trên external network app-net. App kết nối qua host nội bộ shared-mysql và shared-redis.

deploy/proxy

Chạy shared Nginx trên external network proxy-net, nhận traffic HTTP/HTTPS và route domain về từng service.

deploy/book_store

Chạy bookstore-api và bookstore-workers từ image GHCR của backend production branch.
